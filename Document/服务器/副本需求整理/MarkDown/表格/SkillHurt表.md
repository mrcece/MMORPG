# 一.表格说明

- SkillHurt是受击伤害配置表
- 一行一个受击伤害配置,供SkillCollision,FlyBomb表索引使用
- 定义伤害计算相关的属性值

# 二.表格字段说明

- 字段名加*表示有详细说明

| 字段名                | 字段含义            |
| --------------------- | ------------------- |
|ID|唯一id|
|HurtID|受伤id|
|Level|skill等级|
|DamageValue|固定伤害值|
|DamageProportion|基础伤害率|
|ArmorDamageValue|霸体伤害值|
|BreakTime|不用|
